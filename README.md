# RepoPulse

RepoPulse is a simple GitHub portfolio analyzer built with React, TypeScript, and the GitHub REST API.

Users can search for any public GitHub profile and view a clean summary of repositories, stars, forks, top languages, and recently updated projects.

## Features

* Search GitHub users
* Display profile information
* Analyze public repositories
* Show total stars and forks
* List top and recently updated repositories
* Responsive interface

## Tech Stack

* React
* TypeScript
* Vite
* GitHub REST API
* CSS
